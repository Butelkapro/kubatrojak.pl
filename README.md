# kubatrojak.pl 🌐

Moja osobista wizytówka i cyfrowe portfolio, stworzone z naciskiem na minimalizm, wydajność i czytelność.

## 🛠️ Technologie i Narzędzia
Projekt jest rozwijany w podejściu **AI-assisted development**, co pozwala na szybkie prototypowanie i utrzymanie wysokiej jakości kodu.

* **Frontend:** HTML5, CSS3 (Modern Dark Mode), JavaScript (ES6+)
* **Edytor:** [Visual Studio Code](https://code.visualstudio.com/)
* **Version Control:** [GitHub Desktop](https://desktop.github.com/) & Git
* **AI Support:** Google Gemini
* **Hosting:** [GitHub Pages](https://pages.github.com/)

## 📁 Struktura strony
* `/about/` – Informacje o mnie
* `/aboutsite/` – Szczegóły techniczne strony i polityka prywatności
* `/contact/` – Dane kontaktowe i media społecznościowe
* `/go/` – System szybkich przekierowań
* `/globalfiles/` – Zasoby wspólne (ikony, grafiki)

## ⚖️ Prywatność
Strona jest statyczna i nie zbiera danych użytkowników. Szczegóły dotyczące hostingu znajdują się w sekcji [O stronie](https://kubatrojak.pl/aboutsite/).
