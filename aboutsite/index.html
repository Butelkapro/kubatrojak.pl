O stronie

Ostatnia aktualizacja strony: 28 czerwca 2026

Witaj! Ta strona to moja cyfrowa wizytówka i miejsce, gdzie dokumentuję swoje projekty oraz osiągnięcia.

Od strony technicznej, witryna jest rozwijana jako projekt Open Source przy użyciu nowoczesnych narzędzi programistycznych. Kod tworzę w edytorze Visual Studio Code, a do zarządzania wersjami i publikacji wykorzystuję GitHub Desktop. W procesie optymalizacji i projektowania rozwiązań wspieram się sztuczną inteligencją Gemini, co pozwala mi na szybkie wdrażanie nowych funkcjonalności i dbanie o czystość kodu. Całość hostowana jest na platformie GitHub Pages, co zapewnia szybkość i niezawodność działania.

Prawa autorskie

Kod źródłowy strony jest otwarty i udostępniany na licencji MIT. Oznacza to, że możesz go swobodnie kopiować i modyfikować, pod warunkiem zachowania informacji o autorze. Pełna treść licencji z podpisem autora znajduje się w pliku LICENSE w repozytorium projektu na GitHubie.

Polityka prywatności

Twoja prywatność jest dla mnie ważna. Niniejsza strona ma charakter statyczny i nie zbiera, nie przechowuje ani nie przetwarza danych osobowych użytkowników w sposób aktywny.

Informuję jednak, że strona jest hostowana przy użyciu usługi GitHub Pages. Dostawca hostingu (GitHub, Inc.) może automatycznie gromadzić podstawowe dane techniczne (np. adres IP, rodzaj przeglądarki). Szczegółowe informacje na ten temat znajdziesz w Oświadczeniu o prywatności GitHub. Jako właściciel strony nie mam wpływu na te procesy i nie ponoszę odpowiedzialności za działania dostawcy usług.
